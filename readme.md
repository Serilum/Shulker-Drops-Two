<p><strong><span style="font-size:24px">Requires the library mod <a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow">Collective</a>.</span></strong><br><br><strong>&nbsp;&nbsp;&nbsp;This mod is part of <span style="color:#008000"><a style="color:#008000" href="https://curseforge.com/minecraft/modpacks/the-vanilla-experience" target="_blank" rel="nofollow">The Vanilla Experience</a></span> modpack and <span style="color:#804d4f"><a style="color:#804d4f" href="https://curseforge.com/minecraft/mc-mods/serilums-extra-bundle" target="_blank" rel="nofollow">Serilum's Extra Bundle</a></span> mod.</strong><br><span style="font-size:18px">Shulker Drops Two is a minimalistic configurable mod that allows Shulkers to drop two or more shells, up to 64. It'll use the normal drop chance of shells by default. This means that the amount of shells dropped will only be altered if one would've dropped without the mod. You may also enable an option in the configuration page to make Shulkers always drop their shell and ignore the drop chance entirely.</span><br><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/Serilum/.information/wiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span></strong><br><img src="https://github.com/Serilum/.cdn/raw/main/projects/shulker-drops-two/a.png" alt="The Forge Config Manager page." width="666" height="136"></p>
